人生总该那么美好，但是为啥给我遇到cocopods

  • 之前都没有觉得装cocopods那么复杂，找了一堆网上的资料，反正好好把这几天遇到的坑慢慢整理并从ruby开始一直装到cocopods，如果遇到问题的朋友可以看下我之前的文章，我有空在一起整理好，反正我会慢慢更新

  • 注意如果删除了ruby，cocopods等插件都会被删除，所以要重新开始装ruby～

  • 经历了这几天坑爹的安装之旅，大大的感慨一番…
    由于网上面都没有很详细的冲ruby安装到cocopods安装，要不就是都是复制粘贴的坑爹货，
  • 简单说下安装的思路，优胜美地和船长在XCODE安装下自带就有了ruby，我就是蛋疼自己删除了，正常情况下，只要更新下最新的ruby就可以安装cocopods，但是安装完cocopods使用过程会出现一堆坑爹的问题，希望大家耐得住寂寞，面包就会有的了！

安装cocopods思路，自己百度ruby语言和gem的用法

安装流程

  • 以下的方法适用于2016年7月20日

使用方法

//以下是我2016年7月20日安装ruby艰辛的过程，好久没装那么辛苦了！
自己安装流程

1、移除对应版本 rvm remove 1.8.7
2、重装对应版本 rvm reinstall ruby-2.2.4

3、全部删除 rvm implode
4、更新 gem -v
gem update –system
5、清除旧版本gem gem cleanup

列出 gem sources -l
添加镜像 gem sources -a http://ruby.taobao.org/
移除镜像 gem sources -r http://rubygems.org/
安装rack测试 gem install rack

安装rails
安装rails gem install bundler rails
检查bundle路径 bundle -v
检查rails路径 rails -v

Setting up CocoaPods master repo
$ /usr/bin/git pull –ff-only
